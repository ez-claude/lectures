대출이자 계산기의 기본 골격을 만들어줘.

[절대 규칙 — 이 프로젝트 전체에서 지켜야 함]
- Next.js는 이미 설치되어 있다. npx create-next-app 실행하지 마. npm install next 하지 마.
- npm install 금지. 외부 패키지 절대 설치하지 마. 어떤 패키지도 추가 설치 금지.
- tailwind 설치 금지. 순수 CSS만 사용.
- 모든 기능은 React + CSS + 브라우저 내장 API만으로 구현.
- 'use client' 컴포넌트로 만들어.
- 기존 파일(page.tsx 등)을 수정해서 만들어. 새 프로젝트를 생성하지 마.
- 터미널 명령 사용을 최소화해. 파일 수정만으로 해결할 수 있으면 터미널 쓰지 마.
- 웹사이트 검색하지 마. 디자인은 아래 명시된 대로만.

[이번 턴에서 만들 것]
1. 입력 영역: 대출금액(슬라이더+입력), 금리(슬라이더+입력), 기간(프리셋 버튼 5/10/15/20/25/30년)
2. 결과 영역: 월 납입금(크게), 총 상환액, 총 이자
3. 원리금균등 계산만 (원금균등은 다음 턴에 추가)
4. 기본값: 1억원, 4.5%, 30년

[계산]
- 원리금균등: M = P * r * (1+r)^n / ((1+r)^n - 1)
- r = 연이율 / 12 / 100
- 이자 원 미만 버림 (Math.floor). 반올림 금지.

[디자인 — 검색하지 말고 이대로 만들어]
- 배경: #f8f9fa
- 카드: 흰색, border-radius: 16px, box-shadow: 0 2px 8px rgba(0,0,0,0.08)
- 포인트 색상: #3182f6
- 월 납입금 숫자: font-size 36px, font-weight 800
- 여백 넉넉하게. 전체 한글.
- 모바일 대응: 768px 이하에서 세로 스택

다 만들면 "Turn 1 완료" 라고 알려줘. 다음 기능은 내가 말할 때까지 추가하지 마.
