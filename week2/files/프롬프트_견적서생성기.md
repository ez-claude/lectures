# 견적서 자동 생성기 - 프롬프트 템플릿

> 아래 내용을 그대로 복사해서 Antigravity AI 채팅창에 붙여넣으세요.

---

견적서 자동 생성기 웹앱을 만들어줘.

[목적]
엑셀 수작업 견적서를 웹에서 입력하고, 실시간 미리보기로 확인하고, 인쇄/PDF로 바로 내보내는 도구.
서버/DB 없이 브라우저에서만 동작해야 해.

[핵심 UX]
- 2패널 레이아웃: 왼쪽 입력(스크롤) / 오른쪽 A4 미리보기(고정)
- 입력하면 미리보기에 즉시 반영
- 모바일에서는 입력/미리보기 탭 전환

[필수 기능]
1. 발행처(회사명/대표자/사업자번호/주소) + 수신처(회사명/담당자/주소) 입력
2. 품목 추가/삭제: 품목명, 수량, 단가 입력 → 금액 자동 계산
3. 견적번호 자동: QT-YYYY-MMDD-001 형식
4. 유효기간 자동: 발행일 + 30일
5. 항목별 할인율 + 전체 할인율
6. 부가세 자동: 할인 후 금액의 10%, 원 미만 버림(Math.floor)
7. 납품조건 드롭다운 (선입금100%/납품후30일/50:50/협의)
8. 비고란 (직접입력 + "설치비 별도", "운송비 포함" 등 빠른 삽입 버튼)
9. 로고 업로드 → 견적서 상단에 표시
10. 서명/도장 업로드 → 견적서 하단에 표시
11. 인쇄 버튼 → A4 용지에 견적서만 깔끔하게 출력
12. localStorage 자동 저장 (새로고침해도 데이터 유지)

[계산 규칙]
- 품목 금액 = 수량 x 단가
- 소계 = 모든 품목 금액 합
- 전체 할인 = 소계 x 할인율, 원 미만 버림
- 부가세 = (소계 - 전체할인) x 10%, 원 미만 버림 (반올림 금지)
- 총액 = 할인 후 금액 + 부가세

[디자인]
- 깔끔한 비즈니스 스타일. 네이비/다크블루 기조
- A4 미리보기는 실제 종이처럼 그림자 효과
- 금액은 우측 정렬, 굵게
- 입력 영역은 섹션별 카드로 구분
- 전체 한글 UI

[기본값] 앱 처음 열면 샘플 데이터가 채워져 있어야 함:
- 발행처: (주)샘플컴퍼니 / 홍길동
- 품목: 웹사이트 개발 3,000,000원, UI/UX 디자인 1,500,000원

[금지]
- 서버/DB, 로그인, 외부 API, 외부 이미지 URL
