잘 됐어. 이제 기능을 추가해줘. 기존 코드는 유지하고 아래만 추가/수정해.

[추가할 기능]
1. 상환 방식 토글: 원리금균등 / 원금균등 선택. 각각 한 줄 설명 포함
2. 원금균등 계산: 매회 원금 = P/n, k회차 이자 = 잔액 * r, 이자 원 미만 버림
3. 총 상환액 = 각 회차 납입금을 하나하나 더한 합 ("월납입금 x 회차" 금지)
4. 마지막 회차에서 잔액이 정확히 0이 되도록 보정
5. 이자 비율 표시: 총이자 / 총상환액 * 100 (%)
6. 원금 vs 이자 비율: SVG로 도넛 차트 직접 그려. 외부 라이브러리 금지.
7. 숫자 천 단위 콤마
8. 금리 비교: 현재 기준 -0.5% / 현재 / +0.5% 의 월 납입금 3열 비교
9. 상환 스케줄 표: 접기/펼치기. 회차별 납입금, 원금, 이자, 잔액
