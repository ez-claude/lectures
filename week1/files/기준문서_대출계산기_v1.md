# 대출이자 계산기

> 이 문서를 AI에게 그대로 전달하면 대출이자 계산기 웹앱을 만들 수 있습니다.

---

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 이름 | 대출이자 계산기 |
| 형태 | 웹앱 (서버/DB 없이 브라우저에서만 동작) |
| 대상 사용자 | 대출을 고려 중인 일반인 (금융 비전문가) |
| 핵심 목표 | 대출 금액, 금리, 기간을 입력하면 월 납입금과 총 이자를 즉시 보여주는 계산기 |

은행 앱보다 빠르고, 네이버 계산기보다 예쁘고, 엑셀보다 간단해야 한다.

---

## 필수 기능 (10개)

| 번호 | 기능 | 우선도 |
|------|------|--------|
| 1 | 대출금액 입력: 슬라이더 + 직접입력 동시 제공. 범위 500만~10억, 1000만 단위 | 필수 |
| 2 | 금리 입력: 슬라이더 + 직접입력. 범위 1.0~15.0%, 0.1% 단위 | 필수 |
| 3 | 기간 선택: 프리셋 버튼(5/10/15/20/25/30년) + 직접입력(1~40년) | 필수 |
| 4 | 상환 방식 선택: 원리금균등 / 원금균등 토글. 각각 한 줄 설명 포함 | 필수 |
| 5 | 결과 표시: 월 납입금(가장 크게), 총 상환액, 총 이자, 이자비율(%) | 필수 |
| 6 | 도넛 차트: 원금 vs 이자 비율 시각화 | 필수 |
| 7 | 상환 스케줄 테이블: 접기/펼치기. 회차별 납입금, 원금, 이자, 잔액 | 필수 |
| 8 | 금리 비교: 현재 금리 기준 +-0.5% 시 월 납입금 변화 (3열 비교 카드) | 중요 |
| 9 | 모바일 반응형: 데스크탑 2열, 모바일 세로 스택 | 필수 |
| 10 | 숫자 자동 콤마 포맷팅 (천 단위) | 필수 |

### 슬라이더 + 직접입력 동기화 규칙

슬라이더를 움직이면 입력칸 숫자가 바뀌고, 입력칸에 숫자를 치면 슬라이더가 따라 움직여야 한다. 두 값은 항상 같아야 한다.

## 계산 공식

### 원리금균등 상환 (매달 같은 금액)

```
월 이자율(r) = 연 이자율 / 12 / 100
총 회차(n) = 대출 기간(년) x 12
월 납입금(M) = 대출원금 x r x (1+r)^n / ((1+r)^n - 1)
```

- 매월 납입금이 동일
- 초기에는 이자 비중이 높고, 후반에는 원금 비중이 높음

### 원금균등 상환 (매달 줄어드는 금액)

```
매회 원금 상환 = 대출원금 / n
k회차 이자 = (대출원금 - (k-1) x 대출원금/n) x r
k회차 납입금 = 원금 상환 + k회차 이자
```

- 매월 납입금이 점점 줄어듦
- 총 이자가 원리금균등보다 적음

### 반드시 지켜야 할 계산 규칙

| 규칙 | 내용 | 이유 |
|------|------|------|
| 이자 절사 | 원 미만은 버림 (Math.floor). 반올림(Math.round) 금지 | 은행 실무 기준 |
| 총 상환액 | 각 회차 납입금을 하나하나 더해서 합산. "월납입금 x 회차"로 계산하면 안 됨 | 절사 누적으로 차이 발생 |
| 마지막 회차 | 잔액이 0이 되도록 보정 (누적 오차 반영) | 정확성 |
| 금액 표시 | 천 단위 콤마, "원" 단위 | 가독성 |

## 디자인 방향

| 항목 | 지침 |
|------|------|
| 전체 분위기 | 토스(Toss) 스타일. 깔끔하고 현대적인 금융 앱 느낌 |
| 배경 | 밝은 회색 또는 흰색 |
| 카드 | 흰색 카드, 둥근 모서리(border-radius 12~16px), 부드러운 그림자 |
| 포인트 색상 | 파란색 계열 (토스 블루 참고) |
| 월 납입금 숫자 | 크고 굵은 폰트. 한눈에 들어와야 함 |
| 슬라이더 | 부드러운 트랙, 파란색 핸들 |
| 전체 언어 | 한글만 사용. 영어 메뉴/라벨 금지 |
| 여백 | 넉넉하게. 빽빽하면 안 됨 |

## 입력 기본값

앱을 처음 열었을 때 빈 화면이 아니라 바로 결과가 보여야 한다.

| 입력 | 기본값 |
|------|--------|
| 대출 금액 | 1억원 (100,000,000원) |
| 연 이자율 | 4.5% |
| 대출 기간 | 30년 |
| 상환 방식 | 원리금균등 |
