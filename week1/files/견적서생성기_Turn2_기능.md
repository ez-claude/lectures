잘 됐어. 이제 기능을 추가해줘. 기존 코드는 유지하고 아래만 추가/수정해.

[추가할 기능]
1. 품목 추가/삭제: 버튼으로 품목 행 추가, 각 행에 삭제 버튼
2. 견적번호 자동생성: QT-YYYY-MMDD-001 형식 (오늘 날짜 기반)
3. 유효기간 자동: 발행일 + 30일 표시
4. 항목별 할인율 입력 + 전체 할인율 입력
5. 할인 계산: 소계에서 할인 적용 후 부가세 계산. 원 미만 버림(Math.floor)
6. 납품조건 드롭다운: 선입금100% / 납품후30일 / 50:50 / 협의
7. 비고란: 텍스트 입력 + 빠른 삽입 버튼("설치비 별도", "운송비 포함", "AS 1년 포함")
8. 로고 업로드: FileReader API로 base64 변환 → 견적서 상단 표시. 외부 라이브러리 금지.
9. 서명/도장 업로드: FileReader API로 base64 변환 → 견적서 하단 표시
10. 인쇄 버튼: window.print() 사용. @media print CSS로 입력 영역 숨기고 A4 견적서만 출력
11. localStorage 자동 저장: 새로고침해도 데이터 유지

[계산 규칙 — 정확히 따라]
- 품목 금액 = 수량 x 단가
- 소계 = 모든 품목 금액 합
- 전체 할인 = 소계 x 할인율, 원 미만 버림(Math.floor)
- 부가세 = (소계 - 전체할인) x 10%, 원 미만 버림(Math.floor). 반올림 금지.
- 총액 = (소계 - 전체할인) + 부가세
