# 대출이자 계산기 - 프롬프트 템플릿

> 아래 내용을 그대로 복사해서 Antigravity AI 채팅창에 붙여넣으세요.

---

대출이자 계산기 웹앱을 만들어줘.

[목적]
대출 금액, 금리, 기간을 입력하면 월 납입금과 총 이자를 즉시 보여주는 계산기.
서버/DB 없이 브라우저에서만 동작해야 해.

[필수 기능]
1. 대출금액: 슬라이더 + 직접입력 (500만~10억, 1000만 단위). 슬라이더와 입력칸 항상 동기화
2. 금리: 슬라이더 + 직접입력 (1.0~15.0%, 0.1% 단위)
3. 기간: 프리셋 버튼(5/10/15/20/25/30년) + 직접입력
4. 상환 방식: 원리금균등 / 원금균등 토글 (각각 한 줄 설명 포함)
5. 결과: 월 납입금(가장 크게), 총 상환액, 총 이자, 이자비율(%)
6. 도넛 차트: 원금 vs 이자 비율
7. 상환 스케줄 표: 접기/펼치기, 회차별 납입금/원금/이자/잔액
8. 금리 비교: 현재 기준 +-0.5%일 때 월 납입금 변화 (3열 비교)
9. 모바일 반응형 (데스크탑 2열 / 모바일 세로 스택)
10. 숫자 천 단위 콤마

[계산 규칙 - 반드시 지켜야 함]
- 원리금균등: M = P x r x (1+r)^n / ((1+r)^n - 1), r = 연이율/12/100
- 원금균등: 매회 원금 = P/n, k회차 이자 = 잔액 x r
- 이자는 원 미만 버림(Math.floor). 반올림 금지 (은행 실무 기준)
- 총 상환액 = 각 회차 납입금 하나하나 더한 합. "월납입금 x 회차" 금지
- 마지막 회차에서 잔액이 정확히 0이 되도록 보정

[디자인]
- 토스(Toss) 스타일: 밝은 배경, 흰색 카드, 둥근 모서리, 파란 포인트 색상
- 여백 넉넉하게, 월 납입금 숫자는 크고 굵게
- 전체 한글 UI (영어 메뉴/라벨 금지)

[기본값] 앱 처음 열면 이 값으로 결과가 바로 보여야 함:
- 금액 1억원, 금리 4.5%, 기간 30년, 원리금균등

[금지]
- 서버/DB, 로그인, 외부 API, 광고
