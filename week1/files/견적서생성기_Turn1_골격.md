---

견적서 자동 생성기의 기본 골격을 만들어줘.

[절대 규칙 — 이 프로젝트 전체에서 지켜야 함]
- Next.js는 이미 설치되어 있다. npx create-next-app 실행하지 마. npm install next 하지 마.
- npm install 금지. 외부 패키지 절대 설치하지 마. 어떤 패키지도 추가 설치 금지.
- tailwind 설치 금지. 순수 CSS만 사용.
- 모든 기능은 React + CSS + 브라우저 내장 API만으로 구현.
- 'use client' 컴포넌트로 만들어.
- 기존 파일(page.tsx 등)을 수정해서 만들어. 새 프로젝트를 생성하지 마.
- 터미널 명령 사용을 최소화해. 파일 수정만으로 해결할 수 있으면 터미널 쓰지 마.
- 웹사이트 검색하지 마. 디자인은 아래 명시된 대로만.

[이번 턴에서 만들 것]
1. 2패널 레이아웃: 왼쪽 입력(스크롤) / 오른쪽 A4 미리보기(고정)
2. 발행처 입력: 회사명, 대표자, 사업자번호, 주소
3. 수신처 입력: 회사명, 담당자, 주소
4. 품목 1개 입력: 품목명, 수량, 단가 → 금액 자동계산(수량x단가)
5. 결과 영역: 소계, 부가세(10%), 총액 표시
6. A4 미리보기에 입력 내용이 즉시 반영

[계산]
- 품목 금액 = 수량 x 단가
- 부가세 = 소계 x 10%, 원 미만 버림(Math.floor). 반올림 금지.
- 총액 = 소계 + 부가세

[기본값] 앱 처음 열면 샘플 데이터가 채워져 있어야 함:
- 발행처: (주)샘플컴퍼니 / 홍길동 / 123-45-67890 / 서울시 강남구
- 수신처: (주)클라이언트 / 김담당 / 서울시 서초구
- 품목: 웹사이트 개발 / 1식 / 3,000,000원

[디자인 — 검색하지 말고 이대로 만들어]
- 배경: #f5f5f5
- 카드/입력 영역: 흰색, border-radius: 12px, box-shadow: 0 2px 8px rgba(0,0,0,0.08)
- 포인트 색상: #1a237e (네이비)
- A4 미리보기: 흰색 종이 느낌, box-shadow 강하게
- 금액: 우측 정렬, font-weight: 700
- 숫자 천 단위 콤마
- 전체 한글. 여백 넉넉하게.
- 모바일(768px 이하): 입력/미리보기 탭 전환

다 만들면 "Turn 1 완료" 라고 알려줘. 다음 기능은 내가 말할 때까지 추가하지 마.
